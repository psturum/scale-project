Overview of files

leaderboard.py:
- Contain two function, which updates and loads the leaderboard

predict.py:
- Contain a function for transforming pic into prediction of class,
    and a function too convert the food prediction into carbon emission

visualization.py:
- Contain functions for different visualizations

user_interaction.py:
- Contain functions for producing user input

visualization_pics:
- Contain images for visualization

camera_pics:
- Contain temporary images used for input

pics:
- Contain images used for testing

font:
- Contains fonts for visualizations

To run the program enter: python3 main.py
Follow instructions in terminal
